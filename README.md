# Recipe Scaler

A Python tool to easily adjust recipe quantities.

## Quick Start
1. Run: `python recipe_scaler.py`
2. Pick a recipe
3. Enter servings needed
4. Get your scaled recipe

## Features
- Scale any recipe
- Smart unit conversion
- Clean output
